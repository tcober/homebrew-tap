# Tcober Tap

## How do I install these formulae?

`brew install tcober/tap/<formula>`

Or `brew tap tcober/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tcober/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
